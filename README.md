# starling

https://shvn-inna.github.io/starlingweb


## О проекте

Рабочий прототип сайта с простым компонентным дизайном в рамках учебной практики.  
Реализована базовая структура страниц, маршрутизация и модульная архитектура компонентов.  


## что используется

### Frontend
- React
- React Router
- Vite
- Tailwind CSS
- Node.js
- npm
- Git

---
## Текущая реализация

Главная страница (Welcome)
Навигация по разделам, адаптивная верстка
Базовая структура страниц (About, Databases, Resources)
Компонентные блоки и стили для повторного использования и возможности расширения и дополнения сайта

---

## Дальнейшая работа
- Приведение всех компонент к единым стилям
- Переработка информации разделов для актуализации данных на сайте
